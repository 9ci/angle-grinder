Develop Angle Grinder

There are a few easy ways to quickly get started with Angle Grinder

Checkout from GitHub repository

      $ git checkout https://github.com/9ci/angle-grinder.git

Install the latest nodejs version

{% highlight bash %} sudo apt-get install build-essential openssl libssl-dev pkg-config wget http://nodejs.org/dist/v0.10.12/node-v0.10.12.tar.gz tar -xzf node-v0.10.12.tar.gz cd node-v0.10.12 ./configure make sudo make instal {% endhighlight %}

Install required tools and libraries

Install npm packages for coffee-script, grunt-cli, bower and karma
{% highlight bash %} npm install -g coffee-script npm install -g grunt-cli npm install -g bower npm install -g karma {% endhighlight %} Install npm and bower components
{% highlight bash %} npm install bower install {% endhighlight %} Install foreman ruby gem
{% highlight bash %} gem install foreman {% endhighlight %}

Install PhantomsJS and CasperJS for integration testing

Download and install PhantomJS

{% highlight bash %} wget https://phantomjs.googlecode.com/files/phantomjs-1.9.1-linux-x86_64.tar.bz2 tar xvjf phantomjs-1.9.1-linux-x86_64.tar.bz2 cd tar xvjf phantomjs-1.9.1-linux-x86_64 ln -sf `pwd`/bin/phantomjs /usr/local/bin/phantomjs {% endhighlight %} Detailed instructions about PhantomJS installation: http://phantomjs.org/build.html

Checkout and install CasperJS

{% highlight bash %} git clone git://github.com/n1k0/casperjs.git cd casperjs git checkout tags/1.0.2 ln -sf `pwd`/bin/casperjs /usr/local/bin/casperjs {% endhighlight %} Detailed instructions about CasperJS installation: http://casperjs.org/installation.html

Grunt tasks

The list of basic grunt tasks useful during the delopment:
grunt server
Compile assets and start the web server on localhost:9000
grunt build
Build the production minified release of Angle Grinder
grunt clean
Clear all temporary files and folders including the production release

Running tests

grunt coffeling
Run style checker for CoffeeScript source files
grunt test
Run unit tests.
It will also generate istanbul test coverage report inside ./coverate directory.
grunt test:casperjs
Run CasperJS integration tests
Unit test might be also executed directly by karma with the following command karma start test/karma-coffee.conf.coffee.
Karma watches all the files, specified within the configuration file, and whenever any file changes, it triggers the test run.

Example tests execution output: {% highlight bash %} > karma start test/karma-coffee.conf.coffee ................................................................................ ................................................................................ ....
