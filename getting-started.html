Develop Angle Grinder

There are a few easy ways to quickly get started with Angle Grinder

Checkout from GitHub repository

        $ git checkout https://github.com/9ci/angle-grinder.git

Install the latest nodejs version

        
sudo apt-get install build-essential openssl libssl-dev pkg-config

wget http://nodejs.org/dist/v0.10.12/node-v0.10.12.tar.gz
tar -xzf node-v0.10.12.tar.gz

cd node-v0.10.12
./configure
make
sudo make instal

Install required tools and libraries

        
npm install -g coffee-script
npm install -g grunt-cli
npm install -g bower

npm install
bower install
